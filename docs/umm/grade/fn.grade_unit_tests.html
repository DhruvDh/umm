Expand description

Runs mutation tests using to grade unit tests written by students.

  • req_name: display name for requirement to use while displaying grade result
  • out_of: maximum possible grade.
  • target_test: a list of tests to mutation test.
  • target_class: a list of classes that can be mutated.
  • excluded_methods: a list of method names that are excluted from mutation testing.
  • avoid_calls_to: a list of methods to avoid calls to.
