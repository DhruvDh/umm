Struct tabled::style::CustomStyle

source · 
pub struct CustomStyle<Top, Bottom, Left, Right, Horizontal, Vertical, Header> { /* private fields */ }
Expand description

CustomStyle represents a style controlling a valid state of it.

For example. It doesn’t allow to call method CustomStyle::top_left_corner unless CustomStyle::left and CustomStyle::top is set.
