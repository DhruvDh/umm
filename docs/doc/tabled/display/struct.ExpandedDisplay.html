Expand description

ExpandedDisplay display data in a ‘expanded display mode’ from postgress. It may be usefull for a large data sets with a lot of fields.

See ‘Examples’ in https://www.postgresql.org/docs/current/app-psql.html.

It escapes strings to resolve a multi-line ones. Because of that colors may not be rendered.
